Installation Guide

Task 1: Installation of Arduino IDE on Ubuntu VM

1. Install the Arduino IDE on the Ubuntu VM

To install the Arduino IDE, follow these steps:

sudo apt update

 sudo apt install arduino

To install Arduino IDE,use this two commard line.

Screenshot:

2. Open the Arduino IDE and open the Blink sketch from the Arduino Examples

After installation, open the Arduino IDE by type arduino in Terminal. Navigate to File > Examples > 01.Basics > Blink to open the Blink sketch.

Screenshot:

3. Select an Arduino board as the target

Select the board by navigating to Tools > Board > Arduino Uno (or your specific board).

Screenshot:

4. Build the Arduino sketch example

To build the sketch, click on the Verify button (checkmark icon) in the Arduino IDE.

Screenshot:

5. Select the serial port corresponding to the attached Arduino board

Navigate to Tools > Port and select the port to which your Arduino is connected (e.g., /dev/ttyUSB0).

Screenshot:

6. Upload the compiled Arduino sketch to the Arduino board

Click the Upload button (right arrow icon) to upload the sketch to the board.

Screenshot:

Task 2: Installation of VS Code IDE software and extensions

1. Install VS Code IDE for Windows (Host OS)

Download and install Visual Studio Code from the official website.

Software Version: Visual Studio Code 1.XX (or the latest available version)

Screenshot:

2. Install the Remote Development Extension Pack for VS Code IDE

Open VS Code, go to the Extensions view by clicking the Extensions icon or pressing Ctrl+Shift+X, and search for "Remote Development Extension Pack". Click Install.

Screenshot:

3. Connect VS Code to the remote machine (Ubuntu)

Use the Remote-SSH extension to connect to your Ubuntu VM. Open the command palette (Ctrl+Shift+P), type "Remote-SSH: Connect to Host", and enter your SSH connection details.

Screenshot:

4. Install the PlatformIO extension for Remote Development

In the Extensions view, search for "PlatformIO IDE" and click Install.

Screenshot:

5. Create a project using PlatformIO and the Arduino framework

In VS Code, open the PlatformIO Home screen, click on "New Project", and follow the prompts to create a new Arduino project.

Screenshot:

6. Build the Arduino sketch and upload it to the target

In PlatformIO, click the Build button (checkmark icon) to compile the sketch, then click the Upload button (right arrow icon) to upload it to the Arduino board.

Screenshot:

Troubleshooting

Common issues and solutions:

  • If the Arduino IDE fails to install, ensure your package manager is up to date.
  • If VS Code cannot connect to the remote machine, verify your SSH configuration and network settings.
  • For any PlatformIO errors, consult the PlatformIO documentation.

Further assistance can be found on the Arduino Forums and the Stack Overflow community.
